# Blockchain

## General

- [**Bitcoin: A Peer-to-Peer Electronic Cash System**](https://www.bitcoin.com/bitcoin.pdf). *Satoshi Nakamoto*.

## Consensus

- [**The Bitcoin Backbone Protocol: Analysis and Applications**](https://eprint.iacr.org/2014/765.pdf). *Juan Garay, Aggelos Kiayias, Nikos Leonardos*. EUROCRYPT '15.

- [The Bitcoin Backbone Protocol with Chains of Variable Difficulty](https://eprint.iacr.org/2016/1048.pdf). *Juan Garay, Aggelos Kiayias, Nikos Leonardos*. EUROCRYPT '15. CRYPTO '17.

- [Full Analysis of Nakamoto Consensus in Bounded-Delay Networks](https://eprint.iacr.org/2020/277.pdf). *Juan Garay, Aggelos Kiayias, Nikos Leonardos*. 

- [Analysis of the Blockchain Protocol in Asynchronous Networks](https://eprint.iacr.org/2016/454.pdf). *Rafael Pass, Lior Seeman, abhi shelat*. EUROCRYPT '17.

- [Bitcoin as a Transaction Ledger: A Composable Treatment](https://eprint.iacr.org/2017/149.pdf). *Christian Badertscher, Ueli Maurer, Daniel Tschudi, Vassilis Zikas*. CRYPTO '17.

- [SoK: A Consensus Taxonomy in the Blockchain Era](https://eprint.iacr.org/2018/754.pdf). *Juan Garay, Aggelos Kiayias*. RSA '20.

- [Blockchain with Varying Number of Players](https://eprint.iacr.org/2020/677.pdf). *T-H. Hubert Chan, Naomi Ephraim, Antonio Marcedone, Andrew Morgan, Rafael Pass, Elaine Shi*.

- [Ouroboros Genesis: Composable Proof-of-Stake Blockchains with Dynamic Availability](https://eprint.iacr.org/2018/378.pdf). *Christian Badertscher, Peter Gaži, Aggelos Kiayias, Alexander Russell, Vassilis Zikas*. CCS '18.

- [Ouroboros Chronos: Permissionless Clock Synchronization via Proof-of-Stake](https://eprint.iacr.org/2019/838.pdf) *Christian Badertscher, Peter Gaži, Aggelos Kiayias, Alexander Russell, Vassilis Zikas*
